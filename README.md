A responsive quiz application with timer, score tracking, and animations.

## Features

- ⏱️ Timer with countdown
- 📊 Progress tracking
- 🏆 High score system
- 🎨 Animated interface
- 📱 Fully responsive
- 🌓 Dark theme

## Installation

````bash
git clone https://github.com/Youssefhikal93/React-Quiz.git
cd quiz-app
npm install
npm start

Click "Start Quiz"

Answer questions

View results
